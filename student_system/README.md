# 📚 Student Registration & Login System

> A complete web application demonstrating user registration, authentication, and session management using PHP, MySQL, HTML, CSS, and JavaScript.

---

## 📋 Project Overview

This project is a **full-stack student registration and login system** developed as a class assignment. It allows students to create accounts, log in securely, and view their personal dashboard with all registered information.

### 🎯 Purpose

This assignment demonstrates:
- ✅ Web form design and validation
- ✅ Server-side scripting with PHP
- ✅ Database integration with MySQL
- ✅ Session management for user authentication
- ✅ Secure password handling (hashing)
- ✅ Responsive web design with CSS
- ✅ AJAX for smooth form submission

---

## ✨ Features

### Student Features
| Feature | Description |
|---------|-------------|
| 📝 **User Registration** | Create new account with personal details |
| 🔐 **Secure Login** | Login using email and password |
| 👤 **Personal Dashboard** | View all registered information |
| 🎨 **Responsive Design** | Works on desktop, tablet, and mobile |
| ✅ **Form Validation** | Both client-side and server-side validation |
| 🔒 **Session Protection** | Protected pages require login |

### Technical Features
| Feature | Description |
|---------|-------------|
| 🗄️ **MySQL Database** | Stores user data securely |
| 🔑 **Password Hashing** | Passwords encrypted using bcrypt |
| 🌐 **AJAX Requests** | Smooth form submission without page reload |
| 📱 **Mobile Friendly** | Responsive CSS grid and flexbox |
| 🎭 **Error Handling** | User-friendly error messages |
